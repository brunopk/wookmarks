# Wookmarks

Chrome extension to manage bookmarks

## Usage

1. Build the code :

   ```shell
   yarn build
   ```

2. Install generated content in *dist/* folder through *chrome://extensions*

Development with hot reloading :

```bash
yarn dev
```
